# AuraAI Clone - Avatar & Music Video Generator

A web-based application that creates avatars from photos and generates music videos with custom lyrics or scene descriptions, similar to Aura.ai.

## Features

### Avatar Generator
- Upload photos and transform them into various avatar styles
- Multiple avatar styles: Cartoon, Anime, Realistic, Cyberpunk
- Real-time preview and download capabilities
- Canvas-based image processing for unique effects

### Music Video Generator
- Create music videos from lyrics or scene descriptions
- Multiple music styles: Pop, Rock, Electronic, Hip Hop, Classical, Jazz, Ambient
- Various video styles: Animated, Cinematic, Abstract, Lyrics Focused, Storytelling
- Customizable duration (15-180 seconds)
- Option to include generated avatars in videos
- Real-time audio generation using Web Audio API
- Canvas-based video frame generation

### Technical Features
- Pure HTML, CSS, and JavaScript (no external dependencies required)
- Responsive design for mobile and desktop
- Progressive Web App (PWA) ready
- GitHub Pages compatible
- Client-side processing (no server required)
- Web Audio API for music generation
- Canvas API for image and video processing
